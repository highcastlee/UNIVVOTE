# UNIVVOTE
### 기획 배경
- 대학 총학생회에 대한 낮은 선거 투표율과 흥미 부족.

### 목표
- 선거 [실시간 여론 현황] 및 [학과별 투표 약속 TOP10]을 공개하여 흥미를 유발하며 투표를 장려.
- 각 후보별 공약들에 학생들의 [좋아요]를 측정, 개별적인 공약의 선호도를 수치로 보여줌.
